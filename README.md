**Azure Highly Available Web Infrastructure (Terraform)**
**Overview**

This project creates a Highly Available web infrastructure on Azure using Terraform. It sets up core cloud resources like networking, virtual machines, and optional load balancing.

**Features**
1. Azure Virtual Network (VNet)
2. Subnets and Network Security Groups (NSG)
3. Multiple Virtual Machines
4. Infrastructure as Code using Terraform
5. Technologies Used
6. Terraform
7. Microsoft Azure
8. Linux Virtual Machines
9. Azure Networking (VNet, NSG)
